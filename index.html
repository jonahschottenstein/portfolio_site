About

I began studying web development after graduating from college and was immediately hooked. I found it to involve everything I was looking for in a career: problem-solving, creativity, continuous learning, opportunities to tackle challenges both alone and as part of a team, and the ability to see my ideas realized.

I soon went on to complete multiple freecodecamp.org certificates, supplementing that education with web development tutoring and a graphic design specialization on coursera.org. Concurrently, I was taking on individual projects, making sure there were aspects of each project that I had no prior experience with. This practice is most responsible for my growth as a developer, as it helped me reinforce what I was being taught in my courses, learn concepts and technologies that my courses did not cover, and develop a self-confidence that keeps me composed when I'm faced with problems I don't initially know how to solve.

I now believe I have a solid foundation in modern web development practices that, in conjunction with my eagerness to learn and the importance I put on being a positive workplace presence, will enable me to be an asset to a development team.

Crossword Creator

Create a custom crossword puzzle from scratch. Then export the puzzle as a pdf that is formatted according to the New York Times submission guidelines.

Key features:

CBT Toolbox

Based on the Zones of Regulation, a self-regulation framework rooted in cognitive behavioral therapy, children and neurodivergent adults create custom sets of tools to help themselves manage their feelings in trying circumstances

  • A cognitive behavioral therapy toolbox with sections for the Blue Zone, Yellow Zone, and Red Zone, enabling individuals to quickly access the appropriate tools for their current state, making self-regulation easier in difficult situations

Content Blocker

A browser extension that prevents access to webpages whose URLs contain an item from the user-defined list of strings to censor

Built with Chrome APIs to take advantage of Chrome's detailed documentation and useful developer tools

Key features:

  • For each item in the list of strings to censor, enable the user to block or redirect network requests when the URL contains that string
  • Using Chrome's declarativeNetRequest to dynamically prioritize which list item's action is taken when the contents of a web address match multiple items in the list of strings to censor
  • Can create, read, update, and delete list items (the four operations of a CRUD application)
